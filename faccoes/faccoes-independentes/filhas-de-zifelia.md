# Filhas De Zifelia
As Filhas de Zifelia é nome dado às devotas de [Zifelia](/divindades/panteaon-das-treze-estrelas/zifelia#zifelia). Todos os membros são Fêmeas de qualquer uma das raças de Drafeon.
